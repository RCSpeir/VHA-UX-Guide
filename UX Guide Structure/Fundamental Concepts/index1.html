Fundamental Concepts Human-centered design builds on a set of principles that help you create products that put users and their needs first. These concepts create the foundation for the user experience process and provide the underlying reasons for using specific methods. If you're just getting started with the human-centered design and the user-experience process -- or you want a refresher -- start by exploring these topics.
